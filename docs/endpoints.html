Endpoints

Here is a list of the scopes you require to access the API Endpoints and Webhook Events.

Login​

POST/DneroArk/login/otp
POST/DneroArk/login
POST/DneroArk/login/updateToken
POST/DneroArk/logout

User​

GET/DneroArk/user
POST/DneroArk/user/contacts

Transactions​

GET/DneroArk/transactions
GET/DneroArk/transactions/{transactionId}

Coins​

GET/DneroArk/coins
GET/DneroArk/coins/{coinId}
POST/DneroArk/coins/redeem/{coinId}
POST/DneroArk/coins/Drop
GET/DneroArk/coins/pending
GET/DneroArk/coins/pending/count
